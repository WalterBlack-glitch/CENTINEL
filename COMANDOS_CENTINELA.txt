================================================================

[9] SI NADA APARECE EN EL DASHBOARD

- Espera 2-3 min (90s de gracia + 60s de intervalo de rootcheck).
- Si sigue en cero y tu sistema está limpio, ESO ES CORRECTO.

[10] FLAGS ÚTILES RESUMEN
================================================================

--web                        dashboard HTTP
--web-host HOST              bind del dashboard (default 127.0.0.1)
--web-port N                 puerto (default 8787)
--web-token TOKEN            token Bearer (auto si bind != loopback)
--rootcheck                  18 capas anti-rootkit
--rootcheck-interval N       segundos entre escaneos (default 60)
--netwatch                   tracker procesos<->IP
--sniff                      captura de paquetes (necesita root)
--honeypot PUERTOS           honeypot SSH (ej. 2222 o 22,2222)
--respond-live               bloqueo real con nft/iptables
--block-threshold N          score mínimo para bloquear (default 50)
--baseline-dir RUTA          baselines firmadas (HMAC) persistentes
--ack-baseline               borra baselines y refija en próximo arranque
--maintenance-grace N        segundos de gracia tras arranque (default 90)
--maintenance-off            modo paranoico (sin supresión)
--no-drop                    no soltar privilegios (no recomendado)
--force-drop                 soltar aunque haya capas que requieran root
--doctor                     solo diagnóstico, no arranca
--no-doctor                  saltar diagnóstico previo
